# O Alquimista — Prompts de Imagem para Apresentação

> Estilo base para todos os prompts: **Editorial photography, dark moody lighting, rich amber/gold tones, cinematic composition, shallow depth of field, 4K, professional studio quality**

---

## Notas de Estilo

**Paleta de cores consistente:** #0a0a0a (fundo), #d4a85a (dourado), #1a1814 (superfícies), #e8e6e0 (texto claro)

**Ferramentas recomendadas:**
- **Midjourney v6.1** — melhor para fotografia editorial e composições cinematográficas
- **DALL-E 3** — bom para composições conceituais e flat lays
- **Gemini Imagen 3** — bom para fotografia realista com tons quentes

**Dica:** Adiciona --no text, letters, words, writing no Midjourney para evitar texto gerado na imagem.
