# O Alquimista

> A cinematic coffee laboratory focused on ritual, customization and modern alchemy.

## Concept

O Alquimista transforms the act of drinking coffee into a personalized ritual experience.

Users compose their own coffee formula through:
- origin selection
- roast curve
- aromas
- extraction methods

The project combines:
- editorial design
- UX thinking
- cinematic motion
- premium interaction
- ritual storytelling

Inspired by:
- laboratory systems
- modern alchemy
- specialty coffee culture

## Features

- Interactive blend builder
- Responsive design
- Premium motion interactions
- UX-focused interface
- Cinematic storytelling
- Ritual-based navigation

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Figma
